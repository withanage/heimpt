bin.captionclassifier¶

captionclassfier: a tool to search for potential table titles and then link in-text entities

Usage:

captionclassfier.py tables <input> [options] captionclassfier.py graphics <input> [options] captionclassfier.py all <input> [options]

Options:
-d, --debug

Enable debug output

-h, --help

Show this screen.

-v, --version

Show version.
